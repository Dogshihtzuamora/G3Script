Documentação G3script

G3script é uma linguagem para renderização 3D usando WebGL. Permite definir objetos 3D, transformações e configurações de câmera.

Comandos Básicos

Variáveis

vari nome valor;

Define uma variável que pode ser referenciada como $nome em outros comandos.

Importação

IMPORT "caminho/arquivo.g3s"

Importa outro script G3script.

Objetos

Criação de Objetos

ADD ID "nome_objeto":
  // Comandos do objeto
End;

Define um novo objeto 3D com um identificador.

Vértices

V x, y, z

Define um vértice nas coordenadas (x,y,z).

Cores

C r, g, b

Define a cor atual (valores entre 0 e 1).

Faces

F i1, i2, i3 C=#RRGGBB

Cria uma face triangular usando os índices de vértices i1, i2, i3 (começando de 0). A parte C=#RRGGBB é opcional para definir a cor da face.

Transformações

Dentro de um bloco de objeto

R x, y, z

Rotação em graus nos eixos X, Y, Z

O x, y, z

Origem/Translação

S x, y, z

Escala

Fora de um bloco de objeto

MOV x, y, z ID "nome_objeto"

Move um objeto existente

C r, g, b ID "nome_objeto"

Muda a cor de um objeto existente

Câmera

CAM x, y, z LOOK tx, ty, tz

Posiciona a câmera em (x,y,z) olhando para o ponto (tx,ty,tz).

Exemplo

vari altura 2.0;

ADD ID "cubo":
  V -1, -1, -1
  V 1, -1, -1
  V 1, 1, -1
  V -1, 1, -1
  V -1, -1, 1
  V 1, -1, 1
  V 1, 1, 1
  V -1, 1, 1
  
  C 1, 0, 0
  F 0, 1, 2
  F 0, 2, 3
  
  S 1, $altura, 1
End;

CAM 0, 0, 5 LOOK 0, 0, 0
